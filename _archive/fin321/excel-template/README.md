# FIN-321 — Archived Excel starter workbooks (retired)

These are the **v1** FX-hedging Excel starter materials, retired from the live project.
They are kept here for provenance only — **do not distribute to students.**

| File | What it was |
|------|-------------|
| `FIN 321 - Chapter 8 Transaction Hedging_2026_Stauffer_template.xlsx` | The class-provided Chapter 8 transaction-hedging starter workbook (receivable + payable tabs). |
| `FIN321_FX_Hedging_Template.xlsx` | A later skeleton build used during the v1 reorganization. |
| `README-v2-skeleton-plan.md` | The planned progressive-reveal skeleton set (skeleton → partial → full-guided). Never fully produced. |

## Why retired

The **v2 six-stage redesign** removes the student-facing starter template entirely: students
write a specification first (Stage 2) and generate the workbook from it (Stage 3) — *"there is
no starter template; your spec is the template."* The instructor reference / grading key is the
master workbook:

> **`docs/spreadsheets/International Finance Spreadsheets.xlsx`** — Chapter 8
> "Transaction Hedging (Receivable)" and "Transaction Hedging (Payables)" tabs.

Point students to that master workbook as the worked reference, not to anything in this folder.
