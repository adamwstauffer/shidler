# `docs/`

All written deliverables and reference documents live here. Excel and data files belong in `models/` and `data/`, respectively.

## Subdirectories

| Folder | Purpose | Course stage |
|--------|---------|--------------|
| [`decisions/`](decisions/) | Memos and decision documents | Stage 2 |
| [`specs/`](specs/) | Technical specifications | Stage 4 |
| [`plans/`](plans/) | Optional project plans and timelines | Any |
| [`templates/`](templates/) | Pointers to canonical course templates | Reference |

## Naming conventions

- Use lowercase, hyphenated filenames: `fx-hedging-memo.md`, `valuation-spec.md`
- Date-prefix decision memos: `2026-05-09-fx-hedge-decision.md`
- Keep one document per file — split long pieces into linked appendices instead of inflating a single file
