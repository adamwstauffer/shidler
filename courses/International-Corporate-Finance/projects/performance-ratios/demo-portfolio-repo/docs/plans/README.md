# `docs/plans/`

Optional project plans and timelines. Use this folder for documents that describe *future* work — milestones, schedules, scope outlines.

## When to use this folder

- Multi-stage project timelines
- Scope outlines before you start a model build
- Personal study plans for the semester

If you don't have anything plan-shaped, leave this folder empty (the README is sufficient to keep the directory tracked in Git).
