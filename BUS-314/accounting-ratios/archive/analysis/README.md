# Analysis

Store your working analytical files here — spreadsheets, datasets, notebooks, and scripts used during the project.

## Typical Contents

- `.xlsx` / `.csv` — Financial statement data, ratio calculations
- `.py` / `.ipynb` — Python scripts or Jupyter notebooks (if applicable)
- `.md` — Working notes, draft analyses

## Workflow

1. **Gather Data** — Download or copy financial statements from SEC EDGAR, Yahoo Finance, or company IR pages.
2. **Clean & Organize** — Structure data to match named range conventions from your spec.
3. **Compute Ratios** — Build formulas in Excel or code.
4. **Validate** — Check Du Pont decompositions, Balance Sheet balance, and ratio reasonableness.
5. **Export** — Move final deliverables to `../deliverables/`.
